# Adobe Analytics E-Commerce Performance Dashboard

## Overview

This Power BI dashboard provides insights into e-commerce performance using Adobe Analytics data. It helps analyze revenue, orders, customer behavior, marketing channel performance, and customer journey metrics through interactive visualizations and filters.

## Key Metrics

- Total Revenue: 29Bn
- Total Visits: 4M
- Total Orders: 418K
- Average Order Value (AOV): 70K
- Unique Visitors: 3M

## Dashboard Highlights

- Revenue & Order Trend Analysis
- Product Views vs Orders Comparison
- Marketing Channel Performance
- Device Type Distribution
- Customer Activity Funnel
- Cart Abandonment Analysis
- Interactive Quarter, Payment Method & Date Filters

## Key Insights

- Q2 generated the highest revenue.
- Push Notification was the top-performing marketing channel.
- Product P5 received the highest product views.
- Cart abandonment was highest in Push Notification and Paid Search.
- Customer funnel analysis highlighted conversion optimization opportunities.
